IonoPi Python Tool
==========================

Setup
---------------------

  * sudo apt-get install python3-pip
  * pip3 install xxx

  * sudo apt-get install git-core
  * git clone https://github.com/ecometer/arpa_ionopi.git ~/bin/pydas/
  * chmod +x ~/bin/pydas/*

  * python3 ~/bin/pydas/pydas.py
  * python3 ~/bin/pydas/start_pydas.sh
